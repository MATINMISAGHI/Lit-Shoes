<b>Lit Shoes</b>
A modern website that displays shoes available in the inventory either posted by Lit Shoes or by users across the world.

<h4>Stacks Used:</h4>
<ul>
  <li>Node-Geocoder</li>
  <li>JavaScript</li>
  <li>MongoDB</li>
  <li>Mongoose</li>
  <li>Passport</li>
  <li>Express</li>
  <li>Multer</li>
  <li>HTML</li>
  <li>Flash</li>
  <li>Node</li>
  <li>CSS</li>
  <li>JSX</li>
</ul>
